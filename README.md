# Проект: Mesto

## Описание

Учебный проект веб-сервиса Mesto, позволящий пользователям делиться своими фотографиями.
На данный момент реализована функция редактирования профиля.

## Технологии

- Адаптивная верстка;
- Grid lay-out;
- Flex-box;
- Media запросы;
- Java-Script;
